# Zenith Wellness Platform

A modern, highly aesthetic web application targeted at Millennials and Gen Z, focusing on holistic wellness. The platform provides tools for physical health (nutrition and workouts) and a dedicated community space for mental wellness and open discussion.

## Features
- **Aesthetic Premium Design:** Deep dark mode with vibrant glassmorphic elements and fluid gradients for a calming, modern UI.
- **Physical Hub:** Structured workout schedules and macro-friendly recipe cards.
- **Mental Wellness Hub:** Open discussion forums, "Real Life Stories", and options to connect with counselors.
- **Privacy-First:** Toggleable "Post Anonymously" options built into the community feed.
- **No-Build Setup:** Built entirely with vanilla HTML, CSS, and JavaScript. No build step (Node.js/Webpack) required.

## Technologies Used
- HTML5
- CSS3 (Custom Properties, Flexbox, CSS Grid)
- Vanilla JavaScript
- Font Libraries: Google Fonts (Outfit), Phosphor Icons

## Local Development
To run this project locally, simply clone the repository and open `index.html` in any modern web browser, or use a local server:

```bash
# Clone the repository
git clone https://github.com/hygrivakondury/wellness.git

# Navigate to the directory
cd wellness

# Serve locally (Python 3 example)
python3 -m http.server 3000
```
Then visit `http://localhost:3000`.

## Deployment
This project is configured to run flawlessly on GitHub pages since it is a static site. No build process needed. Simply push the files to your repository and enable GitHub Pages from the repository Settings.
